# serial-controlled-led-blink

## Description
This project uses an Arduino Uno and an LED. The user enters a number through the Serial Monitor, and the LED blinks that many times.

## Components
- Arduino Uno
- Breadboard
- LED
- Resistor

## Concepts Learned
- Serial Communication
- User Input
- For Loops

## How It Works
1. The Arduino asks for the number of blinks.
2. The user enters a number in the Serial Monitor.
3. The LED blinks the specified number of times.
